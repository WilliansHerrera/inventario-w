Guía de Implementación desde Cero

Ecosistema de Punto de Venta & Inventario Industrial

1
Requisitos Previos

Antes de comenzar con la instalación, asegúrese de tener configurados los siguientes paquetes en el servidor Windows:

PHP 8.2.x Composer 2.x MySQL 8.0+ Node.js (LTS) Git

2
Configuración del Backend (Laravel)

Abra una terminal en el directorio raíz del proyecto y ejecute secuencialmente:

# 1. Instalar paquetes composer install # 2. Crear archivo de configuración copy .env.example .env # 3. Inicializar claves de encriptación php artisan key:generate php artisan storage:link
Importante: Abra el archivo `.env` recién creado y edite la sección `DB_DATABASE`, `DB_USERNAME` y `DB_PASSWORD` según las configuraciones de su motor MySQL.

3
Inicialización de la Base de Datos

Genere las tablas del sistema y cargue los usuarios administrativos por defecto:

# Ejecutar migraciones y semilla administrativa php artisan migrate --seed

4
Compilación del Cliente de Escritorio (Tauri)

Si requiere desplegar el cliente nativo de Windows para tiendas físicas:

powershell -ExecutionPolicy Bypass -File .\POS-Windows\build.ps1

El instalador definitivo se guardará en `storage/app/public/pos/POS-Setup.msi` listo para distribución.
